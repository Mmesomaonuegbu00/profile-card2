

# Profile Card – Mmesoma Onuegbu

A responsive profile card showcasing my skills, background, and contact information as a Frontend Engineer.

---

## Features

- Clean UI with glassmorphism styling
- Responsive layout
- Dynamic time display (UTC in milliseconds)
- Current day display
- Social media links (GitHub, LinkedIn, Twitter)
- Downloadable CV
- Semantic HTML structure
